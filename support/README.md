# Support

Shell script branch name pattern: `(feature|fix|hotfix)/XXXXXX_description`

- 6-7 digits prefix (issue/ticket number)
- Underscore separator
- Alphanumeric description with hyphens allowed

Valid examples:
- `feature/123456_add-domain-kernel`
- `fix/1234567_fix-context-result`
- `hotfix/123456_urgent-patch`
